Partial Modal Migration

What changed in PF6

The old Modal (with title, variant, actions, footer props) was deprecated and moved to @patternfly/react-core/deprecated. The new composable Modal uses ModalHeader, ModalBody, and ModalFooter sub-components from @patternfly/react-core.

pf-codemods approach: Hybrid pattern

The codemods moved Modal and ModalVariant to the /deprecated import path, but also introduced ModalHeader, ModalBody, and ModalFooter from the new API. The key TabModal component uses a hybrid:

import { ModalBody, ModalFooter, ModalHeader } from '@patternfly/react-core';
import { Modal, ModalVariant } from '@patternfly/react-core/deprecated';

<Modal variant={modalVariant} isOpen={isOpen} onClose={closeModal}>
  <ModalHeader title={headerText} />
  <ModalBody>{children}</ModalBody>
  <ModalFooter>...footer content...</ModalFooter>
</Modal>
