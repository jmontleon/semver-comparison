🔲 EmptyState Component Migration Goose Wins

What Changed in PF6

In PF6, EmptyStateIcon and EmptyStateHeader are internalized into EmptyState. Icons are passed via the icon prop directly on <EmptyState>.

// PF5
<EmptyState variant={EmptyStateVariant.full}>
  <EmptyStateIcon variant="container" component={CubesIcon} />
  <EmptyStateBody>...</EmptyStateBody>
</EmptyState>

// PF6
<EmptyState variant={EmptyStateVariant.full} icon={CubesIcon}>
  <EmptyStateBody>...</EmptyStateBody>
</EmptyState>

🎨 Design Token / Color Migration Goose Wins
