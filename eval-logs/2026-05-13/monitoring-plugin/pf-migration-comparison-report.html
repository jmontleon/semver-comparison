🪟 Modal → Deprecated Migration

What changed in PF6

In PF6, Modal and ModalVariant were moved to @patternfly/react-core/deprecated. They still work from the main export but are officially deprecated. The recommended approach is to either migrate to the new composable Modal pattern or explicitly import from the deprecated path.

import {
  Modal,
  ModalBody,
  ModalFooter,
  ModalHeader,
  ModalVariant,
} from '@patternfly/react-core';

import {
  ModalBody,

import {

} from '@patternfly/react-core/deprecated';

</EmptyState>

🔘 Button Icon Prop Migration

What changed in PF6

In PF6, icon-only buttons should use the icon prop instead of passing the icon as children. This affects buttons with variant="plain" that render only an icon.
