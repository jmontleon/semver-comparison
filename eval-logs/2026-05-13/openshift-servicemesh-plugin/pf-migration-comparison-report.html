fix-engine's unmigrated PfColors.tsx (critical)

The PfColors enum is used throughout the entire Kiali UI for styling. On fix-engine it still references --pf-v5-global--palette--* tokens, which do not exist in PF6 CSS. This means all colors using PfColors will fail to resolve at runtime, leading to invisible text, missing backgrounds, and broken visual styling.
