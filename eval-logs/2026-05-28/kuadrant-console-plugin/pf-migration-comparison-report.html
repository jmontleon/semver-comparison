📋 Modal Component Migration   semver wins

What Changed in PF6

The old Modal (with title prop) was deprecated. PF6 promotes the new Modal from /next which uses <ModalHeader>, <ModalBody>, <ModalFooter> child composition. The old Modal is available via @patternfly/react-core/deprecated as a temporary bridge.
